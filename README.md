source code, sets of one or more MCU program modules together with associated
control data, usage procedures, and operating procedures, are tested to

in the development cycle.

## [](https://github.com/techschool/simplebank#simple-bank-service)Simple bank service

The service that we‚Äôre going to build is a simple bank. It will provide APIs for the frontend to do following things:

1.  Create and manage bank accounts, which are composed of owner‚Äôs name, balance, and currency.
2.  Record all balance changes to each of the account. So every time some money is added to or subtracted from the account, an account entry record will be created.
3.  Perform a money transfer between 2 accounts. This should happen within a transaction, so that either both accounts‚Äô balance are updated successfully or none of them are.
